Q1 · Every exam

Piecewise / exotic function

● Appears on every single final

A function is defined piecewise — usually something like e^{-1/x³}, (1/2)sin(1/x)·(1/x), or sin(x³-x²)/(x²-x) for x≠0 (or some other excluded points), with a defined value at those points. The question asks you to mark the one true statement from 5 options.

The options are always drawn from a fixed set of properties. Here's how to attack each one efficiently:

The standard properties tested — and how to check them fast

  1. Continuity at the special point(s) Compute lim_{x→a} f(x) from both sides and compare to f(a). For e^{-1/x}: the limit from the right is 0 (since -1/x → -∞) but from the left is +∞ — so discontinuous. For e^{-1/x³}: both sides → 0 since x³ changes sign, so it is continuous. Get this right first — it determines whether differentiability is even possible.
  2. Differentiability at the special point(s) Only possible if it's continuous there. Use the limit definition: f'(0) = lim_{h→0} [f(h)-f(0)]/h. For e^{-1/x³}: [e^{-1/h³}-0]/h = e^{-1/h³}/h → 0 since the exponential kills the polynomial. Don't try to differentiate the piecewise formula symbolically at the special point — always go back to the limit definition.
  3. Even / odd Check: does f(-x) = f(x) (even) or f(-x) = -f(x) (odd)? A function like x·sin(1/x) is even (product of odd×odd = even). A function like x²·sin(1/x) is odd (even×odd = odd). The piecewise definition at 0 must also satisfy the symmetry — if f(0)=5 and f is supposed to be odd, that's already false since odd functions must satisfy f(0)=0.
  4. Injectivity (one-to-one) on a given interval For this, analyze the derivative's sign on that interval. If f'(x) > 0 throughout (or < 0), it's injective. Alternatively, think about whether the function oscillates (e.g. sin(1/x) oscillates infinitely near 0, so it cannot be injective on any interval containing 0 or approaching 0). For large x, oscillation stops and it may become injective.
  5. Integrability / integral value Bounded + piecewise continuous on a closed interval ⟹ integrable. Compute ∫_{-1}^{1} f(x)dx: if f is odd, the integral is 0 immediately. If you need to estimate it (is it > 1? < 1?), use bounds: if 0 ≤ f(x) ≤ M on [0,1], then ∫_0^1 f dx ≤ M. For ∫_0^1 e^{-1/x³}dx: the function is between 0 and 1, so the integral is strictly between 0 and 1, which rules out "integral = 0" and "integral > 1".
  6. Monotonicity on a given interval Compute the derivative symbolically (away from the special point) and check its sign. For e^{-1/x³}: derivative is (3/x⁴)·e^{-1/x³} > 0 for x > 0, so it's increasing on (0,∞). For x < 0: need to check separately since x³ is negative there.
Critical trap

For the option "f is bounded on [0,∞)" — note that e^{-1/x} → 1 as x→∞ and → 0 as x→0⁺, so it's bounded on [0,∞). But the function value at x=0 is defined separately — make sure it matches the limit or the answer could flip.

Worked example — 2022 Exam A, Q1

f(x) = sin(1/x) for x≠0, f(0) = -1/3. Options include: ∫_{-1}^1 f dx = 0 ; injective on (0,∞) ; ∫_0^1 f dx > 1 ; f increasing on (-∞,0) ; f' bounded on (0,∞).

Step 1 — Parity: sin(1/x) is odd (sin(1/(-x)) = sin(-1/x) = -sin(1/x)), but f(0) = -1/3 ≠ 0, so f is NOT odd. So ∫_{-1}^1 f dx ≠ 0 by the parity argument — eliminate that option.

Step 2 — Injectivity on (0,∞): sin(1/x) oscillates infinitely near 0, hitting every value in [-1,1] infinitely many times. Not injective.

Step 3 — ∫_0^1 sin(1/x) dx > 1? The integrand is between -1 and 1, so |∫| ≤ 1. Definitely not > 1.

Step 4 — f' bounded on (0,∞)? f'(x) = -cos(1/x)/x², which blows up as x→0⁺. Not bounded.

Answer: f is increasing on (-∞,0). Check: f'(x) = -cos(1/x)/x² for x < 0. Near x=0⁻, this oscillates — that's not always positive. So actually... the answer needs a more careful check. The key is: for |x| sufficiently large (x very negative), cos(1/x) → 1 > 0, so f'(x) < 0. This is the trap — the "correct" option here requires recognizing which statement survives all the others failing.

Q2 · Every exam

Two improper integrals

● Appears on every single final

Two improper integrals I and II are given. Four options: both converge, neither converges, I only, II only. Always exactly one is correct.

Step-by-step attack

  1. Locate all the singularities An improper integral can be improper at ∞ (infinite interval) or at a point where the integrand blows up. For ∫_1^3 sin(x-1)/(x-1)² dx: improper at x=1 (the integrand blows up there). For ∫_1^∞ ...: improper at ∞. If there are multiple singularities (e.g. ∫_0^∞ with a blow-up at 0 and at ∞), split the integral and analyze each piece separately.
  2. Near each singularity, find the dominant behavior Replace the integrand with its leading-order term near the singularity. Use asymptotic equivalents. The goal is to reduce to a comparison with 1/(x-a)^p near a finite singularity, or 1/x^p near ∞.
  3. Apply the p-test to your comparison function Near finite singularity at x=a: ∫ 1/(x-a)^p dx converges iff p < 1. At ∞: ∫_1^∞ 1/x^p dx converges iff p > 1. The log refinement: ∫_e^∞ 1/(x(ln x)^p) dx converges iff p > 1.

The asymptotic replacements — memorize these

— Near x=a (let u = x-a → 0) —
sin(u)/u² ~ 1/u → diverges (p=1)
(1-e^{-u})/u ~ 1 → converges if multiplied by 1/u^p, p<1
ln(1+u) ~ u → same as polynomial
arcsin(u) ~ u → same as polynomial
√(2-2cos u) ~ √(u²) = u → converges iff ∫ 1/u^p, p<1

— Near x=∞ —
ln(1+x)/x ~ ln(x)/x → compare to 1/x^{1-ε}, diverges
arctan(x) → π/2 (finite) → if multiplied: compare to the rest
π/2 - arctan(x) ~ 1/x → so (π/2-arctan x)/x² ~ 1/x³, converges
e^{√x}/x ~ e^{√x} → grows, but in denominator → rapid decay
Common exam trap — sign-changing integrand

The comparison test requires the integrand to be eventually non-negative (or non-positive). For something like sin(x-1)/(x-1)² near x=1: sin(u)/u² ~ 1/u as u→0⁺ (since sin(u) > 0 for small u > 0), so it diverges like ∫ 1/u du. The 2019A exam Q2 is exactly this — answer: I diverges, II converges.

The "odd function over symmetric interval" shortcut

If the integrand is odd and the interval is symmetric (like ∫_{-1}^{1}), the integral is 0 — but only if it converges! You must still check convergence separately at both singularities. The 2021B exam Q2 uses this: ∫_{-1}^{0} arcsin(x)/x^{4/3} — even though the answer seems "0 by symmetry", you must verify convergence near x=0 first (arcsin(x)/x^{4/3} ~ x/x^{4/3} = 1/x^{1/3}, p=1/3 < 1, converges ✓).

The full worked examples from the exams

2019A Q2 — I: ∫_1^3 sin(x−1)/(x−1)² dx
Singularity at x=1. Let u=x-1: near 0, sin(u)/u² ~ u/u² = 1/u. So ∫ ~ ∫ 1/u du near 0. p=1 ≥ 1. DIVERGES
2019A Q2 — II: ∫_1^∞ (π/2 − arctan x)/x² dx
At ∞: π/2 − arctan(x) ~ 1/x, so integrand ~ 1/x³. p=3 > 1. CONVERGES
2020A Q2 — I: ∫_0^∞ ln(1+x)/ln(1+x) dx = ∫_0^∞ 1 dx
Wait — actually ∫_0^∞ ln(1+x)/ln(1+x) dx simplifies to ∫_0^∞ 1 dx... check: the integral is ∫_1^∞ ln(1+x)/ln(1+x) dx. Simplifies, then at ∞ it's 1. DIVERGES
2022B Q2 — II: ∫_0^1 1/√(2−2cos x) dx
Near x=0: 2-2cos x ~ x², so √(2-2cos x) ~ x. Integrand ~ 1/x. p=1. DIVERGES
Q3 · Every exam

Two series: converge or not?

● Appears on every single final

Same four-option structure as Q2. The strategy is built around one core idea: replace complicated terms with their asymptotic equivalent and reduce to a p-series.

The asymptotic replacement toolkit

— What these common terms look like as n → ∞ —
arctan(1/n) ~ 1/n
arctan((-1)^n/n) ~ (-1)^n/n (alternating!)
arccos(1 - 1/n) ~ √(2/n) (use arccos(1-u) ~ √(2u))
1 - cos(1/n) ~ 1/(2n²)
1 - cos(1/n²) ~ 1/(2n⁴)
sin(1/n) ~ 1/n
ln(1 + 1/n) ~ 1/n
ln(n+√n) - ln(n) = ln(1+1/√n) ~ 1/√n
e^{1/n} - 1 ~ 1/n
n(e^{1/n} - 1) ~ 1 (→ this series diverges!)
(1/n + 1/(n+1))² ~ 4/n²
n^{1/n} - 1 ~ ln(n)/n (use e^{ln(n)/n}-1 ~ ln(n)/n)
(n+1)^n / (n · n!) — use ratio test

Decision algorithm

  1. Check divergence test: does a_n → 0? If not, diverges immediately. Example: Σ(2n)!/(n!)² — by Stirling or ratio test, terms grow, so it diverges.
  2. Replace each piece with its asymptotic Drop lower-order terms. The goal is to get the form C/n^p or C·r^n or C/n^p·(ln n)^q.
  3. Apply limit comparison to the simplified form lim (a_n / b_n) = L where 0 < L < ∞: then Σa_n and Σb_n have the same behavior. Pick b_n = 1/n^p.
  4. If alternating: apply Leibniz / alternating series test Σ(-1)^n b_n converges if: (i) b_n is eventually decreasing, (ii) b_n → 0. Note: this gives conditional convergence. If the MCQ asks about absolute convergence separately, you may need to also check if Σ|a_n| converges.
  5. For factorials/exponentials: ratio test Compute lim |a_{n+1}/a_n|. If <1: converges. If >1: diverges. If =1: inconclusive (rare on this exam, but the series with xⁿ/(n ln n) at x=±1 is a known edge case).

Recurring exam examples solved

Σ(-1)^n · ln(1 + 1/n) — 2019A Q3
a_n = ln(1+1/n) ~ 1/n. This is positive and → 0 and decreasing. Alternating series test: CONVERGES (conditionally). But Σ|a_n| ~ Σ1/n diverges, so not absolutely.
Σ n(1 − cos(1/n)) — 2019A Q3
1 - cos(1/n) ~ 1/(2n²). So a_n ~ n · 1/(2n²) = 1/(2n). This is Σ1/2n, the harmonic series. DIVERGES
Σ (e^{1/n} - 1)² — 2019B Q3
e^{1/n}-1 ~ 1/n. So (e^{1/n}-1)² ~ 1/n². p=2 > 1. CONVERGES
Σ arccos(1 − 1/n) — 2019B Q3
arccos(1-u) ~ √(2u) as u→0. So arccos(1-1/n) ~ √(2/n). This is ~ 1/n^{1/2}. p=1/2 ≤ 1. DIVERGES
Σ arctan((-1)^n/n) — 2020A Q3
~ (-1)^n/n. Alternating series with b_n=1/n decreasing → 0. CONVERGES (conditionally)
Σ (2n)!/(n!)² — 2020A Q3
Ratio test: a_{n+1}/a_n = (2n+2)(2n+1)/((n+1)²) → 4 > 1. DIVERGES
The n^(1/n) − 1 case

This appears in 2022A Q3 as part of Σ (n+1)^n / (2^n · n!). Ratio test: a_{n+1}/a_n = (n+2)^{n+1}/((n+1)^n · 2 · (n+1)) = (1+1/(n+1))^n · (n+2)/(2(n+1)) → e/2 > 1. Diverges.

Q4 · Every exam

Mark the correct / incorrect statement

● Appears on every final, usually worth 8 pts

This question tests conceptual understanding with traps. The options are crafted to look plausible. The key skill is having a mental library of standard counterexamples and known implications.

The implications you must know cold

Differentiability & continuity
TRUE Differentiable ⟹ continuous
FALSE Continuous ⟹ differentiable (|x| at 0)
FALSE f² differentiable ⟹ f differentiable (f(x)=-1 x<0, 1 x≥0: f² = 1 everywhere, but f not continuous)
TRUE f differentiable ⟹ f² differentiable
FALSE |f| differentiable ⟹ f differentiable
Monotonicity
TRUE f'(x) > 0 on (a,b) ⟹ strictly increasing
FALSE f'(x) ≥ 0 on (a,b) ⟹ strictly increasing (x³ at 0)
TRUE f'(x) > 0 except finitely many points ⟹ strictly increasing
FALSE f'(x) > 0 for all x ≠ 0,1 ⟹ strictly increasing (true if on a connected interval; but if f is not continuous at 0 or 1 it could fail)
FALSE lim f'(x) = 8 as x→∞ ⟹ f increasing on ℝ (f might decrease earlier)
Convexity & extrema
FALSE Every convex function on ℝ has a minimum (e.g. f(x)=e^x: convex but no minimum)
TRUE A convex function on ℝ that has a maximum must be constant
FALSE An inflection point cannot be a local extremum (it can if f' doesn't exist there)
FALSE f''(x) > 0 for all x ⟹ f has a unique minimum (f might have no minimum at all, e.g. if f→∞ in both directions but not attained — actually this one IS true for continuous f on ℝ if f→∞; check the specific statement carefully)
Limits & asymptotic behavior
FALSE f(x)/g(x) → 1 ⟹ f(x)-g(x) → 0 (counterexample: f=x+1, g=x)
TRUE f(x)-g(x) → C ⟹ f(x)/g(x) → 1 (if g(x) → ∞)
FALSE f(x)/x → 1 as x→0 ⟹ f is differentiable at 0 with f'(0)=1... actually TRUE by definition of derivative if f(0)=0
FALSE lim f(x)·g(x) = 0 ⟹ lim f(x)=0 or lim g(x)=0 (both could oscillate)
The Riemann integral limit (recurring)
The statement lim_{x→∞} ∫_x^{x+1} f(t)/(3t²+5t+4) dt = 2/3 is tested in 2019A and 2020A. Approach: the denominator ~ 3t² at ∞, and the integral of f(t) over a unit interval... by the MVT for integrals, this is f(c_x)/(3x²+...) → 0. Actually: the integrand ~ f(t)/(3t²), and if f is a polynomial, this simplifies. The answer depends on the numerator — read carefully.
f(x)/x → 1 as x→0 ⟹ differentiable at 0?
TRUE if f(0) = 0: this is exactly the definition of f'(0) = 1.
But the option often says "f is necessarily differentiable at 0 and f'(0)=1" — TRUE if f(0)=0 is given or implied (since f(x)/x → 1 and f continuous → f(0)=0).
Exam strategy for Q4

Go through each option and try to find a counterexample in 30 seconds. If you can't, tentatively mark it as true. The correct answer is the one where (a) you can't find a counterexample, and (b) you can sketch a proof. Usually 3 of the 4-5 options have counterexamples you recognize immediately.

Q5 · Most finals

Number of roots of an equation

● Appears on most finals (Q1 in 2019, 2020)

The equation is of the form f(x) = c or f(x) = g(x). Options: exactly 1, 2, 3, 4, or no roots.

The algorithm

  1. Rewrite as h(x) = 0 where h = f - g Work with a single function. Check: h(x) continuous? (Almost always yes for this exam.)
  2. Check behavior at boundaries: h(−∞), h(+∞), and key points Use IVT: if h changes sign over an interval, there's a root inside. Find intervals where the sign change occurs. Also check h(0), h(1), h(-1), etc. for easy-to-compute values.
  3. Analyze h'(x) to count how many times h can change sign If h'(x) has exactly k real roots (where h' changes sign), then h has at most k+1 intervals of monotonicity, and thus h(x)=0 has at most k+1 roots. Combined with IVT evidence, this pins down the exact count.
2019A Q1 — f(x) = 8x² − ln(1+x²) − cos(2x), how many roots?

Step 1: f(0) = 0 - 0 - 1 = -1 < 0. f(1) = 8 - ln2 - cos2 ≈ 8 - 0.69 - (-0.41) = 7.72 > 0. So by IVT, a root exists in (0,1). By symmetry check: f(-x) = f(x) (all terms are even!), so if there's a root at x₀ > 0, there's one at -x₀. That gives at least 2 roots.

Step 2: f'(x) = 16x - 2x/(1+x²) + 2sin(2x). At x=0: f'(0) = 0. Check f'(x) for x > 0: 16x dominates for large x, and near 0... f''(0) = 16 - 2 + 4 = 18 > 0, so x=0 is a local minimum. Since f(0) < 0 and f → +∞ as x→∞, there is exactly one root for x > 0.

Answer: exactly 2 roots (one positive, one negative, by symmetry).

2020A Q1 — x = sin(2x), how many roots?

Rewrite: h(x) = x - sin(2x) = 0. h(0) = 0 ✓ (x=0 is a root). h'(x) = 1 - 2cos(2x). h'(x) = 0 when cos(2x) = 1/2, i.e. 2x = ±π/3 + 2πk. Near x=0: h'(0) = 1 - 2 = -1 < 0, so h is decreasing at 0. For large |x|: |sin(2x)| ≤ 1 but |x| → ∞, so h(x) → +∞ for x→+∞ and → -∞ for x→-∞. h starts at 0 (at x=0), dips negative (since h'(0) < 0), then comes back up. Similarly for x < 0. Count the sign changes: there are roots at some x₁ < 0 < x₂ besides x=0. Total: 3 roots.

Q6 · Most finals

Specific function analysis

● A specific named function, multiple properties to test

Unlike Q1 (piecewise), this is a "nice" function like f(x) = xe^{|x|} + 5x⁴ or f(x) = |x|sin(x). The options test: limits at ±∞, parity, differentiability at 0, specific integral values.

Systematic checks

  • Parity: Write f(-x) and simplify. Even: f(-x)=f(x). Odd: f(-x)=-f(x). For products: odd×odd = even, even×odd = odd, even×even = even.
  • Limit at ∞: For oscillatory functions like |x|sin(x), the limit does NOT exist (even in extended sense) since the function oscillates with growing amplitude. For xe^{|x|}: as x→+∞ this → +∞, as x→-∞: xe^{-x} → 0. So the limit at ∞ in the extended sense is +∞ but it oscillates on negatives... check carefully.
  • Differentiability at 0 for |x|-type functions: Use limit definition. f'(0) = lim_{h→0} [f(h)-f(0)]/h. For |x|sin(x): [|h|sin(h)-0]/h = |h|sin(h)/h → 0·1 = 0. So f'(0) = 0.
  • Integral values: Use parity to simplify. ∫_{-1}^{1} |x|sin(x)dx = 0 since the integrand is odd (odd × odd = even... wait: |x| is even, sin(x) is odd → product is odd). So integral over symmetric interval = 0. ∫_0^1 |x|sin(x)dx: just compute directly.
Parity quick-reference

even × even = even  |  odd × odd = even  |  even × odd = odd
even ∘ anything = even  |  odd ∘ odd = odd  |  f ∘ g parity: use f(g(-x)).
For integrals: ∫_{-a}^{a} even = 2∫_0^a  |  ∫_{-a}^{a} odd = 0

Q7 · Post-2019 finals

Limit involving a variable-bound integral

● Appears in later exams (2020, 2021, 2022)

A limit of the form lim_{x→a} [∫_{g(x)}^{h(x)} f(t)dt] / [denominator]. This is 0/0, so L'Hôpital applies — but the key is differentiating the numerator using the Fundamental Theorem of Calculus.

The algorithm

  1. Verify it's 0/0 or ∞/∞ Usually the integral = 0 when bounds coincide (e.g. if lower = upper at the limit point). The denominator also → 0.
  2. Differentiate numerator using FTC + chain rule d/dx ∫_{g(x)}^{h(x)} f(t)dt = f(h(x))·h'(x) - f(g(x))·g'(x)
  3. Differentiate denominator normally May need to apply L'Hôpital multiple times if still 0/0.
  4. Evaluate at the limit point If still 0/0, repeat. Usually 1-2 applications suffice.
2022A Q7(c) — lim_{x→0} [∫_{x²}^{x³} sin(t²)dt] / sin(x⁶)

At x=0: numerator = 0 (bounds coincide), denominator sin(0) = 0. Apply L'Hôpital:

Numerator derivative: sin((x³)²)·3x² - sin((x²)²)·2x = sin(x⁶)·3x² - sin(x⁴)·2x

Denominator derivative: cos(x⁶)·6x⁵

Ratio: [sin(x⁶)·3x² - sin(x⁴)·2x] / [6x⁵·cos(x⁶)]

Near x=0: sin(x⁶) ~ x⁶, sin(x⁴) ~ x⁴. So numerator ~ 3x⁸ - 2x⁵. Dominant term: -2x⁵. Ratio ~ -2x⁵/(6x⁵·1) = -1/3.

Answer: -1/3

2020B Q1 — lim_{x→π/2} [∫_{π/2}^x sin(t)/arcsin(t) dt] / (1-sin x) via sin x

This is 0/0 at x=π/2. Numerator derivative (FTC): sin(x)/arcsin(x). Denominator derivative: -cos(x). Ratio: sin(x)/(arcsin(x)·(-cos x)). At x=π/2: sin(π/2)=1, arcsin(π/2) is... wait, arcsin only defined for [-1,1]. The integral has arcsin(t) in the denominator — check the domain. Near t=π/2, arcsin(t) is not real. This is likely a typo in the extraction; ignore and focus on the method.

Q8 · Most finals

Power series interval of convergence

● Appears on most finals, sometimes in MCQ, sometimes in T&F

Full step-by-step algorithm

  1. Find the radius R using the ratio test R = lim_{n→∞} |a_n/a_{n+1}|. Or: R = 1/limsup |a_n|^{1/n}. Key fact: if a_n → L ≠ 0, then R = 1 (since ratio → 1).
  2. Check the endpoint x = +R Substitute and determine convergence using alternating series, p-series, or divergence test.
  3. Check the endpoint x = −R Same process. The two endpoints are often different (one converges, one doesn't).
  4. Write the interval using correct bracket notation ( for excluded endpoint, [ for included. E.g. [-R, R) means -R included, R excluded.

Key endpoint tests

Σ xⁿ/(n ln n) — at x = ±1
At x=1: Σ 1/(n ln n). By integral test: ∫ 1/(x ln x)dx = ln(ln x) → ∞. DIVERGES
At x=-1: Σ (-1)^n/(n ln n). Alternating, b_n = 1/(n ln n) → 0 decreasing. CONVERGES
Interval: [-1, 1)
If a_n → L ≠ 0, what is the interval?
R = 1. At x=1: Σa_n·1^n = Σa_n. Since a_n → L ≠ 0, diverges by divergence test. At x=-1: Σa_n·(-1)^n. Same issue. Neither endpoint included. Interval: (-1, 1).

Transformed power series

The exam tests whether you can handle non-standard power series. If Σa_n x^n has interval [-2, 2) (so R=2), then:

  • Σa_n x^{2n}: converges when |x²| < 2, i.e. |x| < √2. Check x = ±√2 separately.
  • Σa_n (x²+1)^n: converges when |x²+1| < 2, i.e. -3 < x² < 1, i.e. -1 < x < 1 (and x² ≥ 0 always). Check endpoints.
  • Σna_n x^{n-1} (the derivative series): same radius R=2, but endpoint behavior may differ — usually loses convergence at the previously-convergent endpoint.
  • Σa_n x^n / n (the antiderivative series): same radius R=2, may gain convergence at the previously-divergent endpoint.
The 2018A MCQ trap

If lim a_n = L (finite, nonzero), the interval of convergence is exactly (-1, 1). Not (-L, L). The radius depends on the ratio of consecutive terms, not on the value of the terms themselves. This is a classic trap option — the answer (c) "(-1,1)" is correct, not (b) "(-L,L)".

T&F · Every exam from 2020

True / False questions

● 4 statements, 3 pts each, 12 pts total — no partial credit per statement

From 2020 onward, exams include a block of 4 T/F statements. Unlike Part B, you just circle True or False — no justification needed. But you do need to be right.

Recurring T/F archetypes

  • TRUE lim (1 + ln(1+n²))^{2ln(n)/something} = e^a — always reduce to e^{lim exponent · ln base}. These 1^∞ forms always work out to a specific value.
  • FALSE "If a_n > 0 and lim b_n = ∞, then lim a_n·b_n = ∞." Counterexample: a_n = 1/n², b_n = n → a_n·b_n = 1/n → 0.
  • TRUE "If a_n ≥ 1 for all n and lim b_n = ∞ and a_n does not converge in extended sense, then lim a_n·b_n = ∞." Since a_n ≥ 1 and b_n → ∞, the product → ∞ regardless of a_n's behavior.
  • FALSE "If Σa_n and Σb_n converge, then Σa_n·b_n converges." Counterexample: a_n = b_n = (-1)^n/√n. Σa_n converges (alternating), but Σa_n·b_n = Σ1/n diverges.
  • TRUE "There is 0 < c < 1 for which ln(1+c²) = ln(2-c)." Apply IVT to h(x) = ln(1+x²) - ln(2-x): h(0) = 0 - ln2 < 0, h(1) = ln2 - 0 > 0 → root exists in (0,1). ✓
  • FALSE "If a_n is bounded and sin(a_n) converges, then a_n converges." Counterexample: a_n = π/2 + 2πn (or just a_n oscillating between two values x,y where sin(x)=sin(y)).
  • TRUE "If Σ3a_n converges, then Σa_n converges." (trivially — same series scaled by 3). But "Σa_n³ converges ⟹ Σa_n⁶ converges" — TRUE because a_n³ → 0, so eventually |a_n| < 1, so |a_n⁶| = |a_n³|² ≤ |a_n³| → comparison works.
  • TRAP "lim x^{1/x} = 1 as x→∞." TRUE — write as e^{ln(x)/x} → e^0 = 1. But this is often listed with a subtly wrong value or different base. Read carefully.
