# Calc 1M - Technion Exam Study Guide
A self-study reference for Technion's Calculus 1M (104018), built by analyzing
patterns across finals and midterms from 2017–2022.

## What's in here

`Chadvaa_1M_Exam_Guide.html` — an interactive study guide covering:

- Exam structure and time allocation
- The 6 MCQ archetypes that appear on every final
- Strategies for limits, sequences, series, Taylor approximations, integrals, and proofs
- A formula sheet add-on with the 10 things most sheets are missing

## How to use it

Just open the HTML file in a browser — no dependencies, no build step.

## Notes

The guide deliberately excludes Lagrange error bound questions, ε-δ proofs,
and theorem-statement-from-memory tasks, since those are covered separately.
